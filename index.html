Previsão de Vendas de Uma Rede de Lojas

Eu usei python, estatística e técnicas de regressão para prever o valor monetário de vendas que cada loja da rede irá produzir nos próximos 6 meses, esse modelo seria usado para os tomadores de decisão saberem quanto poderiam investir em cada loja para melhorias, aumento de estoque, pessoal, etc, tendo como base a entrada financeira prevista no modelo. Além disso, esse modelo foi conectado a um bot no aplicativo de mensagens Telegram, e o gestor pode através de uma simples mensagem receber a previsão de venda da loja que deseja consultar, de qualquer lugar e a qualquer momento.

  • Git and GitHub
  • Python, Pandas, seaborn
  • Sklearn, Random Forest
  • Render Cloud
  • More

Priorização de Clientes Para Cross Sell (Em Construção)

Usei algoritmo supervisionado de classificação para ordenar uma lista de clientes de uma seguradora com os clientes com maior potencial de compra de um segundo produto da empresa. O modelo foi vinculado a uma planilha google e as pessoas da área comercial poderiam com um clique ordenar a lista e ter mais conversão nas vendas. Se esse modelo fosse utiizado geraria uma ordenação ao menos 2,5x melhor do que um método aleatório e com 60% da base conseguiria atingir 100% das pessoas interessadas, gerando economia de recurso e ganho de efetividade.

  • Python, pandas, matplotlib, seaborn
  • LightGBM Classifier
  • Boruta, Lift Curve
  • More
